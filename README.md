# <h1 align="center"> 🚗 C++ Life Board Game Recreation! 🎲 <h1>
### *C++ Recreation of the Game of Life*

---

## 📌 Overview

This project is a terminal-based recreation of the classic board game *Life*, built in C++. It simulates player decisions, random events, and progression through the game using object-oriented design.

---

## 🎮 How to Run / Play

### 1. Download the Project
- Click **Code → Download ZIP**
- Save as `Life.zip`

### 2. Extract Files
- Unzip the file
- Place the folder anywhere on your computer

### 3. Open in a C++ IDE
Use any IDE/editor that supports modern C++:
- Visual Studio  
- Visual Studio Code  
- CLion  

### 4. Ensure C++17 is Enabled
This project requires **C++17 or newer**.

Minimum compiler versions:
- GCC ≥ 7  
- Clang ≥ 5  
- Visual Studio ≥ 2017  

Compile manually (if not using an IDE):
```bash
g++ -std=c++17 Life.cpp -o game
```

### 5. Run the Game
- Open the project folder in your IDE  
- Run `Life.cpp`  
- The game will start in the terminal  

### 6. Play!
- Follow the on-screen instructions  
- Make decisions and progress through the game  

### 7. (BONUS)
- Run the secondary file (will be worked on)
- Input custom data to improve your experience!
